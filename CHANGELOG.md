# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-10

### Added
- Established release governance with mandatory unit, integration, and contract test gates.
- Added packaging smoke checks for wheel and sdist artifacts.
- Added behavior-change enforcement to require tests for modified runtime behavior.

### Changed
- Promoted SyncCraft package version from `0.1.0` to `1.0.0` for first stable release.
- Hardened CLI/test documentation for release and troubleshooting workflows.

### Security
- Documented API-key handling and secret-management best practices for provider configuration.
